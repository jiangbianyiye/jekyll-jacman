### Chapter 01
#### 插入排序

```c
void insertionSort(int* a,int n){
  for(int j=1;j<n;j++){
    int key=a[j];
    int index=j;
    for(int i=j-1;i>=0;i--){
      if(a[i]>key){
        a[i+1]=a[i];
        index=i;
      }
    }
    a[index]=key;
  }
  return;
}
```

#### 练习

1.1-4 How are the shortest-path and traveling-salesman problems given above similar? How are they different?

相同点：最短路径与旅行商问题都要在一张图中找出最短路径；不同点：最短路径只需找出两点间的最短路，而旅行商问题需要找出经过多点并返回原点的最短路径。
> 旅行商问题是NP完全的，没有已知的有效算法。
